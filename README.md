# Tubitak-GITAOS (Smart Parking System)

Bu proje, TÜBİTAK projesi kapsamında geliştirilen görüntü işleme tabanlı otopark sisteminin mobil istemcisidir.

## Teknoloji (Legacy Stack)
Bu proje öğrenme sürecimin başlarında geliştirildiği için **Klasik Android (XML)** mimarisi kullanılmıştır.
* **Dil:** Kotlin
* **Networking:** Retrofit & Gson
* **UI:** XML Layouts & RecyclerView

## Not
Bu proje **akademik bir prototip** olup geliştirme süreci tamamlanmıştır.
Güncel projelerimde (örn: **Aether AI**) modern Android geliştirme pratikleri olan:
* **Jetpack Compose**
* **MVVM Architecture**
* **Clean Architecture**
teknolojilerini aktif olarak kullanmaktayım.
